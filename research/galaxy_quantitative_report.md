## 一、产品定位

银河量化（GalaxyQuantitative，galaxyquantitative.com）是一套 **跨交易所、跨数据源的"无分润"加密合约智能跟单系统**。核心卖点：

- **极致低延迟**：宣称毫秒级响应，交易员 100 单 2 秒内全部跟完
- **无分润机制**：交易员盈利全归用户，按"月租"订阅而非分润
- **隐藏持仓也能跟**：能跟踪私域、未上架、满员、隐藏持仓的带单员
- **跨交易所**：Binance、OKX(欧易)、Gate.io、Bitget；信号源横跨 6 个：币安官方带单员、OKX 官方带单员、币Coin、Bitget、聪明钱链上地址、Hyperbot
- **科学仓位管理**：固定金额 / 全仓 / 复利滚动 + 倍率、安全垫、止盈止损
- **多账号**：单账户最多见 4 个子账号（账户1-4），每个子账号绑一个交易所

### 3.4 启动跟单配置弹窗（最关键 ⭐）

完整参数清单（全部抓取自页面）：

**资金管理模式**（三选一）
- 固定金额
- 全仓跟单
- 复利滚动 —— 以当前账户总资产为基数实时计算仓位

**基础设置**
- **跟单倍率**：与交易员仓位百分比的乘数（1 = 与交易员相同百分比）
- **启动跟单设置**：不复制 / 仅复制浮亏持仓 / 复制所有持仓
- **持仓方向限制**：无限制 / 只开多单 / 只开空单
- **开仓触发条件**：市价 / 持仓均价限价 / 加仓价限价 + 价格优于交易员的百分比 %
- **加仓触发条件**：市价 / 持仓均价限价 / 加仓价限价 + 价格优于交易员的百分比 %

**止盈止损（风控）**
- 持仓止盈：不启用 / 循环触发 / 平仓数量 %
- 持仓止损：不启用 / 循环触发 / 平仓数量 %
- **跟单亏损阈值**：USDT 绝对值；达到阈值时暂停此交易员的跟单并全平来自此交易员的持仓
- **安全垫亏损值** ×：净值跌破预设值时自动触发倍率降低，防止进一步亏损
- 触发止盈后市场价格回到开仓均价时重新补满仓位
- 补满仓位后允许再次止盈

**币种过滤**
- 币种黑名单（不跟单的币种）
- 只跟单的币种（白名单）
- 输入支持联想，逗号分隔

**通知设置**
- 通知方式：不通知 / 邮件 / TG机器人
- 通知类型：下单成功 / 下单失败 / 触发风控 / 止盈止损 / **交易员保证金变动**（仅限币安）
- 风控设置针对来自此交易员跟单任务下的 **所有配置** 生效

操作：取消 / 启动跟单

### 3.5 智能跟单 - 交易所自选

用户自行添加任意 ID（非平台上架的）：
- 分类：全部 / 币安 / 聪明钱 / Hyperbot
- 模式切换：**全域跟单** / **Cookie跟单**（SVIP）
- 搜索框：输入交易员链接 / 身份 ID / 昵称
- 注意：
  - 全域跟单 + Binance 公开持仓 = 无延迟
  - 隐藏持仓 = 2~3 分钟延迟
  - Cookie 跟单需 SVIP 资格（应该是上传交易员官网 cookie 实现实时抓取）

### 3.6 智能跟单 - 币Coin 自选

需先登录用户的 **币coin 账号**（账户/密码）才能抓数据。
注意事项："因币coin平台自身因素，在行情波动剧烈时跟单延迟将不可控；若交易员已隐藏持仓及操作，则无法跟单，并且买卖模式交易员无法爬取操作记录，仅支持爬取持仓信息。"
